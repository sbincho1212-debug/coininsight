## 콜드월렛이란?

기본 개념부터 이해해봅시다.

### 정의

콜드월렛은 인터넷과 완전히 분리된 오프라인 저장 방식입니다.

**핵심:**
- 개인키를 오프라인에 보관
- 인터넷 연결 없음
- 해킹 불가능

**종류:**
1. 하드웨어 지갑 (가장 일반적)
2. 페이퍼 지갑
3. 에어갭 컴퓨터

### 콜드월렛 vs 핫월렛

**핫월렛 (Hot Wallet):**
- 인터넷 연결됨
- 예: 메타마스크, 거래소 지갑
- 편리함
- 해킹 위험

**콜드월렛 (Cold Wallet):**
- 인터넷 분리됨
- 예: Ledger, Trezor
- 불편함
- 해킹 불가능

**비유:**
- 핫월렛 = 지갑 (일상 사용)
- 콜드월렛 = 금고 (장기 보관)

### 작동 원리

**거래 서명 프로세스:**

1. 컴퓨터에서 거래 생성
2. 하드웨어 지갑으로 전송
3. 하드웨어 내부에서 개인키로 서명
4. 서명된 거래를 컴퓨터로 전송
5. 블록체인에 브로드캐스트

**핵심:**
- 개인키가 하드웨어 밖으로 나가지 않음
- 컴퓨터가 해킹당해도 안전
- 물리적 확인 필수

### 왜 필요한가?

**금액 기준:**
- $1,000 이하: 핫월렛 OK
- $1,000-$10,000: 콜드월렛 권장
- $10,000 이상: 콜드월렛 필수
- $100,000 이상: 멀티시그 콜드월렛

**기간 기준:**
- 단기 트레이딩: 핫월렛
- 장기 보유 (6개월+): 콜드월렛

**리스크 허용도:**
- 리스크 회피형: 콜드월렛
- 리스크 중립형: 분산 (80% 콜드, 20% 핫)

## Ledger 초기 설정 가이드

실전으로 시작해봅시다.

### 1단계: 구매 및 개봉

**주의사항:**
- ⚠️ 공식 사이트에서만 구매 (ledger.com)
- ⚠️ 중고 절대 금지
- ⚠️ 아마존, 이베이 조심 (가짜 가능)
- ⚠️ 봉인 확인

**개봉 확인:**
- 박스 봉인 상태
- 제품 시리얼 번호
- 부속품 확인 (케이블, 리플렛)

### 2단계: Ledger Live 설치

**다운로드:**
1. ledger.com/ledger-live 접속
2. 운영체제 선택 (Windows, Mac, Linux)
3. 다운로드 및 설치

**주의:**
- 공식 사이트만 사용
- 가짜 앱 주의

### 3단계: 디바이스 초기화

**연결:**
1. USB 케이블로 컴퓨터 연결
2. Ledger 전원 켜짐
3. "Welcome" 화면

**PIN 설정:**
1. "Set up as new device" 선택
2. PIN 코드 입력 (4-8자리)
3. 재입력 확인

**PIN 팁:**
- 8자리 권장
- 추측 불가능한 번호
- 기억 가능한 번호
- 생일, 전화번호 금지

### 4단계: 시드 문구 백업 (가장 중요!)

**시드 문구란:**
- 24개 영어 단어
- 지갑 복구의 마스터 키
- 절대 유출 금지

**백업 과정:**
1. "Write down your recovery phrase" 선택
2. 화면에 단어 하나씩 표시
3. 포함된 종이에 순서대로 기록
4. 24개 모두 기록
5. 확인 (무작위 단어 입력)

**주의사항:**
- 🚨 절대 디지털로 저장 금지 (사진, 파일 등)
- 🚨 클라우드 금지
- 🚨 이메일 금지
- ✅ 손으로 종이에 기록
- ✅ 2-3부 복사본 작성
- ✅ 다른 장소에 보관

### 5단계: Ledger Live 연동

**연결:**
1. Ledger Live 실행
2. "Add account" 클릭
3. 코인 선택 (예: Bitcoin)
4. Ledger에서 Bitcoin 앱 설치
5. 계정 생성 완료

**여러 코인 추가:**
- Ethereum, Cardano 등
- 각각 앱 설치 필요
- Ledger Live에서 관리

### 6단계: 첫 입금

**테스트 송금:**
1. 소액 먼저 (0.001 BTC 등)
2. 주소 확인 (Ledger 화면과 일치)
3. 전송
4. 수신 확인

**본격 송금:**
- 테스트 성공 후
- 전액 이동
- 주소 재확인 필수

## 시드 문구 보안 전략

가장 중요한 부분입니다.

### 기본 전략: 다중 장소 보관

**방법:**
1. 원본: 집 금고
2. 복사본 1: 은행 안전금고
3. 복사본 2: 부모님 집 (봉인 봉투)

**장점:**
- 화재, 도난 대비
- 분실 위험 최소화

**단점:**
- 여러 장소 관리 필요

### 고급 전략 1: 분산 보관

**방법:**
- 24개 단어를 나눔
- 예: 1-12번 한 장소, 13-24번 다른 장소

**장점:**
- 한 장소 도난 시 안전

**단점:**
- 두 장소 모두 필요
- 복잡함

### 고급 전략 2: Cryptosteel

**제품:**
- 금속 백업 장치
- 화재, 수해 방지
- 영구 보관

**사용법:**
1. Cryptosteel 구매 ($100-200)
2. 시드 문구 각인
3. 금고 보관

**장점:**
- 불에 안 탐
- 물에 안 손상
- 영구적

**단점:**
- 비용
- 도난 시 즉시 노출

### 고급 전략 3: Shamir Backup (Trezor)

**개념:**
- 시드를 여러 조각으로 분할
- M-of-N 복구 (예: 3-of-5)

**예시:**
- 5개 조각 생성
- 3개 조각만 있으면 복구
- 2개 이하로는 복구 불가

**장점:**
- 매우 안전
- 유연한 복구

**단점:**
- Trezor Model T만 지원
- 복잡함

### 절대 하지 말아야 할 것

**❌ 디지털 저장:**
- 휴대폰 메모
- 컴퓨터 파일
- 클라우드 (구글 드라이브 등)
- 이메일

**❌ 공유:**
- 타인에게 말하기
- SNS 게시
- 메신저 전송

**❌ 단일 장소:**
- 집에만 보관
- 화재, 도난 시 끝

**✅ 올바른 방법:**
- 손으로 종이에 기록
- 여러 장소 분산
- 물리적 보안
- 정기 확인

## 콜드월렛 사용 실전

일상적인 사용 방법:

### 입금 (Receive)

**과정:**
1. Ledger Live 실행
2. "Receive" 클릭
3. 코인 선택 (BTC, ETH 등)
4. Ledger에서 주소 확인
5. 주소 복사
6. 거래소/다른 지갑에서 전송

**주의:**
- 반드시 Ledger 화면에서 주소 확인
- 주소 일치 여부 확인
- 클립보드 해킹 대비

### 출금 (Send)

**과정:**
1. Ledger Live 실행
2. "Send" 클릭
3. 코인 선택
4. 수신 주소 입력
5. 금액 입력
6. 수수료 선택
7. Ledger에서 확인 및 서명
8. 전송

**주의:**
- 수신 주소 재확인
- Ledger 화면에서 반드시 확인
- 테스트 송금 권장

### DeFi 사용

**Ledger + 메타마스크:**

**연결:**
1. 메타마스크 설치
2. "Connect Hardware Wallet"
3. Ledger 선택
4. 계정 연동

**사용:**
- 유니스왑, 에이브 등 DeFi 접속
- 메타마스크 연결
- 거래 시 Ledger에서 서명

**장점:**
- DeFi 편의성 + 콜드월렛 보안

### 여러 계정 관리

**이유:**
- 용도 분리 (트레이딩, 장기 보유)
- 프라이버시
- 리스크 분산

**방법:**
1. Ledger Live에서 "Add account"
2. 같은 코인 여러 계정 생성
3. 각각 다른 주소

**예시:**
- BTC 계정 1: 장기 보유
- BTC 계정 2: 트레이딩

## 복구 및 문제 해결

문제 상황 대처법:

### Ledger 분실/파손

**복구 과정:**

**1단계: 새 Ledger 구매**
- 공식 사이트
- 같은 모델 권장

**2단계: 복구 모드**
1. 새 Ledger 연결
2. "Restore from recovery phrase" 선택
3. PIN 설정
4. 시드 문구 24개 입력
5. 복구 완료

**3단계: 계정 확인**
- Ledger Live에서 계정 추가
- 잔액 확인

**소요 시간:** 30분

### 시드 문구 분실

**불행히도:**
- 복구 불가능
- 영구 손실

**예방:**
- 여러 장소 백업
- 정기 확인 (연 1회)
- 가족에게 위치 알림 (봉인)

### PIN 잊음

**해결:**

**3회 실패:**
- Ledger 초기화됨
- 시드 문구로 복구 필요

**대처:**
- 침착하게 기억
- 시도 전 시드 문구 확인
- 3번째는 신중히

### 펌웨어 업데이트

**정기 업데이트:**
- Ledger Live 알림
- 보안 패치
- 새 코인 지원

**과정:**
1. Ledger Live 알림 확인
2. "Update firmware" 클릭
3. Ledger에서 확인
4. 자동 업데이트
5. 재시작

**주의:**
- 공식 앱에서만 업데이트
- 가짜 업데이트 알림 주의
- 잔액 변화 없음 (블록체인에 저장)

## 보안 모범 사례

최고 수준의 보안:

### 1. 구매 단계

**공식 사이트:**
- ledger.com
- trezor.io
- 절대 타 사이트 금지

**중고 금지:**
- 조작 가능성
- 시드 문구 노출

### 2. 설정 단계

**환경:**
- 신뢰하는 컴퓨터
- 공공 와이파이 금지
- 보안 소프트웨어 최신

**시드 문구:**
- 손으로 종이에
- 카메라 꺼두기
- 혼자 있을 때

### 3. 사용 단계

**주소 확인:**
- 항상 Ledger 화면 확인
- 컴퓨터 화면만 보지 말기
- 주소 처음 4자리 + 마지막 4자리

**피싱 방지:**
- 이메일 링크 클릭 금지
- 시드 문구 요구 시 100% 사기
- Ledger는 절대 시드 문구 요구 안 함

### 4. 보관 단계

**Ledger:**
- 사용 후 USB 분리
- 금고 보관
- 접근 제한

**시드 문구:**
- 방화, 방수 용기
- 여러 장소
- 정기 확인

### 5. 상속 계획

**가족에게:**
- Ledger 위치
- PIN (봉인 봉투)
- 시드 문구 위치 (봉인)
- 사용 설명서

**유언:**
- 암호화폐 보유 명시
- 복구 절차 기록
- 신뢰하는 사람 지정

## 콜드월렛 vs 멀티시그

더 높은 보안을 원한다면:

### 멀티시그 콜드월렛

**개념:**
- 여러 개의 Ledger
- 2-of-3, 3-of-5 등
- Gnosis Safe 또는 Casa

**장점:**
- 최고 수준 보안
- 한 Ledger 분실해도 OK
- 내부 배신 방지

**단점:**
- 복잡함
- 비용 (여러 Ledger)
- 설정 어려움

**적합:**
- $100,000 이상
- 기업 자금
- DAO 재무

### 단일 콜드월렛

**장점:**
- 간단
- 저렴
- 충분한 보안

**적합:**
- 개인 투자자
- $100,000 이하

## 자주 하는 실수

피해야 할 함정들:

### 실수 1: 시드 문구 디지털 저장

**사례:**
- "편하게 구글 킵에 저장..."
- 클라우드 해킹
- 전액 도난

**올바름:**
- 손으로 종이에

### 실수 2: 중고 구매

**사례:**
- "새 것처럼 저렴하게..."
- 조작된 펌웨어
- 시드 문구 전송

**올바름:**
- 공식 사이트만

### 실수 3: 주소 확인 안 함

**사례:**
- 컴퓨터 화면만 보고 전송
- 클립보드 해킹 (주소 변경)
- 해커 주소로 전송

**올바름:**
- 항상 Ledger 화면 확인

### 실수 4: PIN 공유

**사례:**
- 가족에게 PIN 알려줌
- 가족 실수로 분실

**올바름:**
- PIN은 본인만
- 상속용은 봉인 봉투

### 실수 5: 백업 안 함

**사례:**
- "나중에 하지 뭐..."
- Ledger 고장
- 시드 문구 못 찾음
- 영구 손실

**올바름:**
- 설정 즉시 백업
- 여러 장소

## 마무리하며

콜드월렛은 암호화폐 보안의 골드 스탠다드입니다.

핵심 요약:

1. 콜드월렛 = 오프라인 저장 = 해킹 불가능
2. $10,000 이상은 필수
3. Ledger vs Trezor: 둘 다 우수
4. 시드 문구가 가장 중요 (종이에 손으로)
5. 여러 장소 백업 필수
6. 주소는 항상 Ledger 화면 확인
7. 공식 사이트에서만 구매
8. 정기 펌웨어 업데이트

시작 가이드:
- 예산형: Ledger Nano S+ ($79)
- 범용: Ledger Nano X ($149)
- 보안: Trezor Model T ($219)

콜드월렛은 초기 투자가 필요합니다.

하지만 수백, 수천만 원을 보호한다면 $100-200는 아깝지 않습니다.
